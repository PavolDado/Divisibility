# Divisibility
Module: divisors.py
Describtion: Function find_divisors(number) returns a list of divisors
of entered number. List starts with 1 and ends with entered number. 

Usage: 
    find_divisors(5) 
    returns 
    [1, 5]
    
    find_divisors(8) 
    returns 
    [1, 2, 4, 8]
